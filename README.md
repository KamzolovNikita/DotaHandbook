# DotaHandbook
Приложение, которое выполняет функцию справочника для игры Дота 2. 
Экраны: 
- Список команд
- Информация о команде
- Список предметов
- Информация о предмете 
- Список героев
- Диалог фильтрации героев
Технологии:
- Dagger 2
- Kotlin Coroutines
- AndroidX Navigation
- OkHttp, Retrofit
- Room
- KotlinXSerialization
- Glide
Архитектура:
- MVVM
- Multi-Module
